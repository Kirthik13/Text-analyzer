
# Text Analyzer

A real-time web application that processes and analyzes the input text.


## Tech Stack
 HTML , CSS , Javascript , Node.js

  
## Features

- Spell Checker 
- Sentiment analysis
Gives real-time count of :-
- Word count
- Character count
- Punctuation count
- Space count
- Sentence count
